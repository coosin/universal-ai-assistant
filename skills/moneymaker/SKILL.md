# Money Maker Skill - 赚钱助手

## 功能

- 分析赚钱机会（在线接单、内容创作、数据标注等）
- 执行自动化任务（需配合 scripts/moneymaker 下脚本）
- 收益监控与报告（cron + message）

## 使用示例

- "请帮我去想办法挣点零花钱"
- "分析一下当前可行的赚钱方式"
- "帮我设置一个每日收益检查任务"

## 工具

- exec / bash：运行脚本
- browser：网页自动化
- cron：定时任务
- web_search / web_fetch：调研机会
- message：结果反馈

## 安全

- 高风险操作（如交易）需用户确认后再执行
