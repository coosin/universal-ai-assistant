## 五、建议执行的操作（可选增强）

若希望进一步提升安全与可维护性，可执行：

```bash
# 1. 密码改用环境变量（可选）
openclaw config get gateway.auth.password   # 先记下当前密码
# 在 systemd 服务中加 Environment=OPENCLAW_GATEWAY_PASSWORD=<密码>
openclaw config unset gateway.auth.password
systemctl --user daemon-reload && systemctl --user restart openclaw-gateway

# 2. 定期安全审计（建议每月一次）
openclaw security audit --deep

# 3. 配置异常时
openclaw doctor --fix
```
